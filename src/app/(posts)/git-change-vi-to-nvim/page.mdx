# Git 커밋 메시지 편집기로 vi 대신 nvim 사용

`git commit`를 할 때

`git commit -m ""`은 제목만을 적게된다

본문도 적고싶다면

`git commit` 명령어로 나오는 `vi 편집기` 화면에서 수정할 수 있다

```bash
git config --global core.editor "nvim"
```

nvim이 설치되어있다면 위 명령어로 Git 기본 에디터를 vi에서 nvim으로 바꿀 수 있다

설정이 잘 되었는지 확인하려면

```bash
git config --global --get core.editor
```

결과값이 `nvim`이 나와야함

- ai
- 2
- d

# dsd
## asd
### asd
